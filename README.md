# nick-does-dev

Personal website built as an interactive terminal experience.

Instead of traditional page sections, visitors type commands and get output directly in a terminal-style UI.

## Tech stack

- React + TypeScript
- Vite
- styled-components

## Terminal commands

Available commands include:

- `help` - list commands
- `whoami` - short bio/about section
- `skills` - skills grouped by category
- `experience` - work history
- `contact` - contact links
- `ls` - fetch and print the current repository tree from GitHub
- `cat <file>` - fetch and print a file from GitHub
- `date` - current date/time
- `pwd` - current origin
- `echo <text>` - echo text back
- `clear` - clear terminal output

There is also some easter eggs command in the app :O

## Getting started

### Prerequisites

- Node.js 20+ (recommended)
- npm

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Then open the local URL shown in the terminal (usually `http://localhost:5173`).
